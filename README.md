Weather App:

A dynamic weather application built with React and Material UI that fetches live data from the OpenWeatherMap API. Search any city worldwide to instantly view current temperature, humidity, and weather conditions — with a UI that visually adapts to the forecast, swapping backgrounds and icons to reflect hot, cold, or rainy conditions in real time.

Key Features :

Live weather data for any city via the OpenWeatherMap API
Condition-aware UI — background imagery and icons shift dynamically based on temperature and weather state
Graceful error handling for invalid or non-existent city searches
Responsive, card-based layout powered by Material UI components

Tech Stack :

React · Material UI · Vite · OpenWeatherMap API
